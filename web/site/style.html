Logo

The logo should always be used over a neutral, light (no darker than #fafafa) background. White (#ffffff) is ideal.

Grid

Pages are divided into grids of 15 for desktop width (greater than 976px), 9 for tablet width (under 976px), and 6 for mobile width (under 630px). With the exception of the spreadsheet, all blocks of content should use the grid.

Typography

Body text uses Lucida sans. Chrome currently misrenders bold styles on Lucida sans, and the difference between normal and bold weight is often negligible. Considering switching to Open Sans or other san serif fonts. Default body font size is 16px, and small text is 12px.

Header text uses Verdana (see header above). Size used around the site is 32px. The CSS is in the process of being consolidated to match that (view-title and panel-title can be the same? consolidate .smaller since most headers if not all use .smaller
